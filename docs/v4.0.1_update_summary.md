## 核心变更对照表

| 维度 | v3.0 / v4.0 | v4.0.1 |
|------|-------------|--------|
| **架构模式** | 纯 P2P 网络 | 双模式（个人版 + 共享版） |
| **身份系统** | Ed25519 密钥对 | BIP39 助记词 → Ed25519 → Node ID |
| **密钥类型** | 6+ 种（含 mk_*） | 4 种（mk_* 废弃） |
| **加密模型** | E2EE | 传输路径加密（中继不可见，资源节点解密） |
| **积分系统** | Contribution | Contribution Credit（不可提现/交易） |
| **开关设计** | 单一开关 | 两级开关（network_enabled + share_to_pool） |
| **公共体验** | 无 | Public Global Key（四重限速） |
| **路线图** | 3 阶段 | 4 阶段（Phase 0/1/2/3） |
| **防共谋** | 基础 | 公证人去中心化演进 |
| **request-id** | 可选 | 缺失 = 不计入积分 |
